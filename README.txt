# Pac-Man Game

## Description
Un projet simple du jeu Pac-Man en utilisant la bibliothèque XamGraph et SDL2. 
Le joueur peut se déplacer dans une grille et collecter des points roses.
Le jeu se termine le joueur clique sur le bouton "Exit".

## Dépendances
- XamGraph : Bibliothèque graphique pour la gestion des fenêtres et des dessins.
- SDL2 : Bibliothèque pour la gestion des événements et des entrées utilisateur.

## Instructions de compilation
1. Assurez-vous que les bibliothèques XamGraph et SDL2 sont installées sur votre système.
    (voici les commande dans le cas ou vous les les auriez pas :

        - sudo apt install libsdl2-dev
        - sudo apt install libsdl2-image-dev
        - sudo apt install libsdl2-ttf-dev
    
    )

2. Utilisez le Makefile fourni pour compiler le programme : make
